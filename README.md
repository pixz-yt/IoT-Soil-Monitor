# 🌡️ IoT Temperature & Humidity Monitor

An IoT monitoring system built using ESP8266 and DHT11. The device measures temperature and humidity and displays the data on a real-time web dashboard.

---

## Features

- Real-time temperature monitoring
- Real-time humidity monitoring
- ESP8266 WiFi connectivity
- Live web dashboard
- Responsive interface

---

## Hardware Used

| Component | Quantity |
|------------|------------|
| ESP8266 NodeMCU | 1 |
| DHT11 Sensor | 1 |
| Breadboard | 1 |
| Jumper Wires | As Required |

---

## Technical Specifications

| Item | Value |
|--------|--------|
| Controller | ESP8266 NodeMCU |
| MCU | ESP8266EX |
| Sensor | DHT11 |
| WiFi | 802.11 b/g/n |
| Operating Voltage | 3.3V |
| IDE | Arduino IDE |

---

## Pinout Table

| Device | ESP8266 Pin |
|----------|------------|
| DHT11 DATA | D2 (GPIO4) |
| DHT11 VCC | 3.3V |
| DHT11 GND | GND |

---

## Wiring

DHT11

VCC → 3.3V

GND → GND

DATA → D2 (GPIO4)
