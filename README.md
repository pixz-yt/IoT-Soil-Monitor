# 🌱 IoT Soil Monitor

Smart IoT-based soil moisture monitoring system using ESP8266 with a real-time dashboard.

## Features

- Real-time soil moisture monitoring
- ESP8266 WiFi connectivity
- Live dashboard
- Sensor data visualization

## Hardware

- ESP8266 NodeMCU
- Soil Moisture Sensor
- Breadboard
- Jumper Wires
